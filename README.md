# AutoSay

A World of Warcraft addon that automatically sends greetings and farewells in party, raid, and guild chat.

## Features

- **Per-channel settings**: Configure separately for Party, Raid, and Guild
- **Customizable triggers**: Choose when to send messages (on join, when others join, on leave)
- **Message pools**: Select from preset greetings/farewells or add custom messages
- **Player names**: Optionally include joining player names in greetings
- **Test mode**: Test all functionality without actually sending messages
- **Cooldown system**: Prevent message spam with configurable cooldown

## Usage

- `/autosay` or `/as` - Open settings
- `/as toggle` - Enable/disable addon
- `/as testmode` - Toggle test mode
- `/as test party` - Simulate joining a party
- `/as test raid` - Simulate joining a raid
- `/as test leave` - Simulate leaving group
- `/as test guild` - Simulate guild login greeting
- `/as test guildbye` - Simulate guild logout farewell
- `/as test player [name]` - Simulate player joining
- `/as status` - Show current status

## Installation

Install via CurseForge or manually copy to your AddOns folder.
